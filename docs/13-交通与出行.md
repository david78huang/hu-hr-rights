## 🚇 布达佩斯公共交通 (BKK)

### 交通方式

| 方式 | 标识 | 说明 |
|-----|------|------|
| **地铁** | M1/M2/M3/M4 | 最快捷，覆盖主要区域 |
| **电车** | 1-69路 | 地面交通，风景好 |
| **公交** | 7-299路 | 覆盖全城 |
| **无轨电车** | 70-80路 | 环保、安静 |
| **郊区火车** | HÉV | 连接布达佩斯与郊区 |
| **渡轮** | D系列 | 多瑙河观光 |

### 地铁线路

| 线路 | 颜色 | 主要站点 |
|-----|------|---------|
| **M1** | 黄色 | 英雄广场、歌剧院、火车站 |
| **M2** | 红色 | 国会大厦、布达城堡、南站 |
| **M3** | 蓝色 | 市中心、机场大巴 |
| **M4** | 绿色 | 多瑙河两岸、南站 |

### 车票类型

| 类型 | 价格（2026） | 适用 |
|-----|-------------|------|
| **单程票** | 450福林 | 单次乘车 |
| **10次票** | 4000福林 | 多次使用 |
| **24小时票** | 2500福林 | 游客推荐 |
| **72小时票** | 5500福林 | 短期停留 |
| **7天票** | 4950福林 | 周票 |
| **月票** | 9500福林 | 长期居住 |
| **学生月票** | 3450福林 | 全日制学生 |

### 购票方式

**1. 自动售票机**
- 地铁站、主要站点
- 接受现金、银行卡
- 有英文界面

**2. 手机App**
- BudapestGO（官方）
- 购买电子票
- 扫码乘车

**3. 便利店**
- 标有BKK标志的店铺
- 如：Relay, Inmedio

### 使用规则

- **上车/进站必须打票** - 否则视为逃票
- **单程票限时** - 地铁80分钟，地面交通80分钟
- **换乘规则** - 单程票可换乘，需在时限内
- **查票频繁** - 便衣查票员随机检查

**逃票罚款：** 8000-16000福林

## 🚗 买车与租车

### 买车流程

```
1. 确定预算
   ├── 车价
   ├── 过户费（约5-10万福林）
   ├── 保险
   └── 维修保养

2. 选车看车
   ├── 二手车网站：hasznaltauto.hu
   ├── 实地看车
   └── 专业检测（建议）

3. 过户登记
   ├── 双方前往车管所（Okmányiroda）
   ├── 签署买卖合同
   ├── 缴纳过户费
   └── 办理新行驶证

4. 购买保险
   ├── 强制险（Kötelező）
   └── 商业险（Casco，建议）
```

### 养车成本（年）

| 项目 | 费用 |
|-----|------|
| 强制险 | 30,000-80,000福林 |
| 商业险 | 100,000-300,000福林 |
| 车检 | 20,000福林/2年 |
| 停车费 | 视区域而定 |
| 油费 | 约600-700福林/升 |
| 保养维修 | 视车况 |

### 租车

**租车公司：**
| 公司 | 特点 |
|-----|------|
| **Hertz** | 国际品牌 |
| **Avis** | 网点多 |
| **Europcar** | 服务好 |
| **Budget** | 性价比高 |

**租车条件：**
- 年龄：通常21岁以上
- 驾照：国际驾照或欧盟驾照
- 信用卡：押金担保

**费用参考：**
- 经济型：15,000-25,000福林/天
- 中型：25,000-40,000福林/天

## 🚂 长途交通

### 火车 (MÁV)

**主要线路：**
- 布达佩斯 ↔ 维也纳：2.5小时
- 布达佩斯 ↔ 布拉格：7小时
- 布达佩斯 ↔ 慕尼黑：7小时

**购票：**
- 网站：https://www.mavcsoport.hu/
- 车站售票处
- 自动售票机

**车票类型：**
- 二等座：标准
- 一等座：更宽敞
- 卧铺：夜间列车

### 长途大巴 (FlixBus/Volanbusz)

**优势：**
- 价格便宜
- 覆盖城市多
- 有WiFi

**购票：**
- FlixBus App/网站
- Volanbusz 网站
- 车站现场

## 💡 出行小贴士

1. **买月票最划算** - 经常乘坐公共交通买月票
2. **下载BudapestGO** - 电子票方便，避免排队
3. **注意查票** - 便衣查票员频繁，务必打票
4. **避开高峰** - 7-9点、17-19点地铁拥挤
5. **夜间交通** - 部分线路有夜班车（900系列）
6. **机场交通** - 100E机场大巴最经济，约2200福林
