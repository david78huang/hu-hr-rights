## 📝 租房流程

```
1. 确定需求
   ├── 预算（租金+杂费）
   ├── 区域（通勤、生活便利）
   ├── 房型（单间/一居/两居）
   └── 租期（短租/长租）

2. 搜索房源
   ├── 网站筛选
   ├── 联系房东/中介
   └── 预约看房

3. 实地看房
   ├── 检查房屋状况
   ├── 了解周边环境
   ├── 询问杂费情况
   └── 确认家具电器

4. 签订合同
   ├── 仔细阅读条款
   ├── 确认押金金额
   ├── 明确维修责任
   └── 双方签字

5. 入住准备
   ├── 缴纳押金和首月租金
   ├── 办理地址卡变更
   ├── 开通水电网
   └── 检查房屋并拍照留证
```

## 🏠 实地看房检查清单

### 房屋状况

- [ ] 墙壁有无裂缝、霉斑
- [ ] 地板是否完好
- [ ] 门窗能否正常开关、密封
- [ ] 暖气是否正常（冬季必查）
- [ ] 热水供应是否稳定

### 设施检查

- [ ] 炉灶/烤箱是否正常
- [ ] 冰箱是否制冷
- [ ] 洗衣机是否正常
- [ ] 灯具是否全部可用
- [ ] 插座是否通电

### 周边环境

- [ ] 交通便利性（公交/地铁距离）
- [ ] 超市、药店距离
- [ ] 周边噪音情况
- [ ] 安全状况（夜间照明等）
- [ ] 停车便利性（如有车）

### 费用确认

- [ ] 租金是否包含物业费
- [ ] 水电暖气费用大致多少
- [ ] 网络费用及速度
- [ ] 垃圾费、电视费等杂费

## 📄 租房合同要点

### 合同必备条款

| 条款 | 说明 | 注意事项 |
|-----|------|---------|
| **租金金额** | 每月租金 | 明确币种（福林/欧元） |
| **支付方式** | 现金/转账 | 转账需保留凭证 |
| **支付时间** | 每月几号 | 通常提前1个月 |
| **押金金额** | 通常为1-2个月租金 | 法律规定不超过3个月 |
| **租期** | 起止日期 | 明确续约条款 |
| **终止条款** | 提前多久通知 | 通常为30天 |
| **维修责任** | 房东/租客分工 | 大修通常房东负责 |

### 合同陷阱警示 ⚠️

**1. 口头承诺不写入合同**
- 所有承诺必须书面化
- 家具清单附在合同后

**2. 押金退还条件模糊**
- 明确什么情况下扣押金
- 明确退还时间和方式

**3. 自动续约条款**
- 注意是否有自动续约
- 如需退租提前通知

**4. 费用承担不清**
- 明确物业费、水电费承担方
- 大修责任归属

**5. 合同语言问题**
- 要求英文或中文翻译件
- 匈语合同以匈语为准，但需理解内容

## 💰 押金规则

### 押金标准

- **法律规定**：不超过 **3个月租金**
- **市场惯例**：通常为 **1-2个月租金**

### 押金用途

1. 房屋损坏赔偿
2. 欠缴租金抵扣
3. 清洁费用（如退房未清洁）

### 押金保护

**存入第三方账户（推荐）**
- 部分正规中介提供押金托管服务
- 退租时双方确认后退还

**直接给房东**
- 要求开具收据
- 拍照/视频记录房屋初始状态
- 退房时对照检查

### 押金退还流程

```
1. 提前通知退租（按合同约定，通常30天）
2. 彻底清洁房屋
3. 修复正常损耗外的损坏
4. 结清所有账单
5. 与房东共同检查房屋
6. 签署退房确认书
7. 退还钥匙
8. 房东退还押金（通常7-30天）
```

## ✅ 入住检查清单

### 入住当天

- [ ] 与房东共同检查房屋状况
- [ ] 拍照/录像记录所有房间、家具
- [ ] 记录电表、水表、气表读数
- [ ] 确认所有钥匙（大门、信箱、储物间等）
- [ ] 获取物业联系方式
- [ ] 获取垃圾投放时间和地点信息

### 入住后一周内

- [ ] 办理地址卡变更（到新地址）
- [ ] 开通/过户水电账户
- [ ] 预约网络安装
- [ ] 购买必要保险（可选）
- [ ] 熟悉周边设施（超市、药店、医院）

## 🚪 退房流程

### 提前准备

1. **提前通知**
   - 按合同约定时间提前通知房东
   - 书面形式（邮件/短信留证）

2. **房屋清洁**
   - 彻底清洁所有房间
   - 清洗窗帘、地毯（如合同要求）
   - 清理冰箱、烤箱

3. **修复损坏**
   - 修复非正常损耗的损坏
   - 保留维修收据

### 退房当天

- [ ] 与房东共同检查房屋
- [ ] 对照入住时照片确认状况
- [ ] 记录最终电表、水表、气表读数
- [ ] 结清所有账单
- [ ] 归还所有钥匙
- [ ] 签署退房确认书
- [ ] 确认押金退还方式和时间

### 押金纠纷处理

如房东无理扣押金：
1. 协商沟通
2. 寻求调解（消费者保护协会）
3. 法律途径（小额法庭）

## 🆘 常见问题

### Q: 没有地址卡可以租房吗？
**A:** 可以，但签订租房合同后需要尽快办理地址卡变更。

### Q: 短租（3-6个月）容易找吗？
**A:** 相对困难，大多数房东偏好长租（1年起）。可考虑Airbnb转长租或学生公寓。

### Q: 租房需要担保人吗？
**A:** 通常不需要，但部分房东可能要求。留学生可能需要学校证明。

### Q: 可以养宠物吗？
**A:** 必须在看房时明确提出，并在合同中注明。多数房东不接受宠物。

### Q: 暖气费很高吗？
**A:** 取决于房屋保温状况和供暖方式。老建筑冬季暖气费可能达200-400€/月。

### Q: 遇到房东恶意扣押金怎么办？
**A:** 
1. 保留所有证据（照片、合同、沟通记录）
2. 向消费者保护协会投诉（NFH）
3. 寻求法律援助
4. 小额法庭起诉（10万福林以下免费）

## 💡 租房小贴士

1. **旺季避开** - 8-9月学生开学季房源紧张、价格偏高
2. **实地看房** - 不要只看照片，必须实地查看
3. **邻居打听** - 了解房屋真实状况（隔音、暖气等）
4. **保留证据** - 所有沟通、转账保留记录
5. **谨慎转账** - 确认房东身份后再支付押金
6. **检查房东** - 要求查看房产证或委托出租证明
