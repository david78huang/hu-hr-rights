# 离职与赔偿完全指南

> 匈牙利辞职、解雇、离职手续办理详细流程

---

## 1. 辞职流程 (Felmondás)

### 1.1 决定辞职前的检查清单

**合同检查**：
- [ ] 查看合同中的通知期条款
- [ ] 确认是否有竞业限制
- [ ] 检查是否有培训赔偿条款
- [ ] 确认年假余额

**财务准备**：
- [ ] 计算最后工资到账时间
- [ ] 确认社保缴纳至何时
- [ ] 准备 1-2 个月生活费

**下家准备**：
- [ ] 新工作 offer 是否已确定
- [ ] 新工作入职时间是否协调好

### 1.2 辞职步骤

#### 第一步：撰写辞职信

**必备要素**：
1. 明确的辞职意向
2. 最后工作日期
3. 通知期说明
4. 年假抵扣请求（可选）
5. 工作交接意愿

**时间线示例**：
```
3 月 1 日：提交辞职信
3 月 1-30 日：30 天通知期
3 月 31 日：最后工作日

或（用年假抵扣）：
3 月 1 日：提交辞职信，申请用 15 天年假抵扣
3 月 1-15 日：实际工作
3 月 16-31 日：休年假
3 月 31 日：最后工作日
```

#### 第二步：发送辞职信

**推荐方式**（保留证据）：
1. **电子邮件** → 发送给直属经理 + HR
2. **纸质版** → 要求签收（Átvételi elismervény）
3. **挂号信** → 如担心公司拒收

**邮件模板**：
```
收件人：经理、HR
抄送：自己（保留记录）

主题：辞职通知 - [你的姓名]

尊敬的 [经理姓名]：

本人特此书面通知，决定于 [最后工作日期] 辞去在 [公司名称] 的 [职位] 职务。

根据劳动合同和匈牙利劳动法，本人将遵守 [30] 天的通知期。

本人希望使用 [X] 天未休年假抵扣部分通知期，如公司同意，实际最后工作日为 [日期]。

请确认收到本辞职信，并告知：
1. 工作交接安排
2. 未休年假结算方式
3. 最终工资发放时间
4. 离职证明开具流程

感谢公司在本人任职期间提供的机会。

此致
敬礼

[姓名]
[日期]
```

#### 第三步：等待确认

**公司应在 3 个工作日内**：
- 书面确认收到辞职信
- 说明离职手续流程

**如公司不回应**：
- 保存发送记录
- 继续正常上班
- 视为公司默认接受

#### 第四步：工作交接

**交接清单**：
- [ ] 归还公司设备（电脑、手机、门禁卡等）
- [ ] 移交工作文件和账号权限
- [ ] 完成交接文档
- [ ] 获得交接确认书

#### 第五步：办理离职手续

**最后工作日当天**：
1. **签署离职文件**：
   - 工作交接确认书
   - 设备归还清单
   - 保密协议提醒（如有）

2. **领取文件**：
   - 离职证明 (Munkáltatói igazolás)
   - 最终工资单
   - 社保缴纳证明（如需）

3. **确认事项**：
   - 最终工资发放日期
   - 年假折算金额
   - 社保缴纳截止日期

### 1.3 辞职后的注意事项

**社保连续性**：
- 确保新雇主及时接续社保
- 间隔期不超过 30 天（影响医疗）

**居留许可**：
- 如持工作居留，需在 30 天内找到新工作
- 或申请其他类型居留

**税务**：
- 次年 3-5 月做年度汇算
- 可能有退税

## 2. 被解雇时的应对

### 2.1 收到解雇通知的第一时间

**保持冷静，做以下事情**：

1. **要求书面通知**
   - 口头解雇无效
   - 必须有书面解雇理由

2. **不要立即签署任何文件**
   - 仔细阅读所有内容
   - 有权带回家考虑 3 天

3. **保留所有证据**
   - 解雇通知书
   - 过往工资单
   - 工作评价/邮件记录

4. **计算应得权益**
   - 法定赔偿金额
   - 未休年假折算
   - 最后工资

### 2.2 解雇合法性检查

**检查清单**：

| 检查项 | 合法 | 可能违法 |
|-------|------|---------|
| 解雇理由 | 有明确书面理由 | 无理由或理由模糊 |
| 提前通知 | 提前 30 天通知 | 当天通知立即走人 |
| 赔偿支付 | 按工作年限支付 | 拒绝支付赔偿 |
| 程序合规 | 有书面记录 | 仅口头通知 |
| 特殊保护期 | 非保护期内 | 病假/孕期/育儿假期间 |

### 2.3 不同情况下的应对策略

#### 情况 A：合法解雇 + 正常赔偿

**特征**：
- 有合理理由（如岗位取消）
- 提前 30 天通知
- 愿意支付法定赔偿

**建议**：
- 接受解雇
- 确认赔偿金额正确
- 协商离职日期（可用年假抵扣）

#### 情况 B：合法解雇 + 赔偿不足

**特征**：
- 解雇理由成立
- 但赔偿计算有误

**建议**：
- 书面指出赔偿计算错误
- 要求按正确标准支付
- 如拒绝，向 OMMF 投诉

#### 情况 C：违法解雇

**特征**：
- 无理由或虚假理由
- 未提前通知
- 拒绝支付赔偿
- 特殊保护期内解雇

**建议**：
1. **立即行动**：
   - 不签署任何文件
   - 继续正常上班（如可能）
   - 发送邮件质疑解雇合法性

2. **收集证据**：
   - 解雇通知书
   - 过往工作评价
   - 同事证言

3. **寻求帮助**：
   - 咨询律师
   - 向 OMMF 投诉
   - 30 天内起诉

4. **争取权益**：
   - 恢复工作 + 补发工资
   - 或更高赔偿（2-5 倍法定标准）

## 3. 离职手续清单

### 3.1 员工需完成

**文件签署**：
- [ ] 工作交接确认书
- [ ] 设备归还清单
- [ ] 保密协议提醒（如有）

**物品归还**：
- [ ] 公司电脑/手机
- [ ] 门禁卡/钥匙
- [ ] 工作服/工牌
- [ ] 公司车辆（如有）

**账号移交**：
- [ ] 邮箱账号
- [ ] 系统账号
- [ ] 客户资料

### 3.2 公司需提供

**必须提供**：
- [ ] 离职证明 (Munkáltatói igazolás)
- [ ] 最终工资单
- [ ] 赔偿支付（如适用）

**建议索取**：
- [ ] 工作推荐信
- [ ] 社保缴纳记录
- [ ] 税务证明

### 3.3 离职证明 (Munkáltatói igazolás)

**内容必须包括**：
1. 雇主信息（名称、地址）
2. 员工信息（姓名、职位）
3. 工作期限（入职/离职日期）
4. 工资水平（用于新雇主参考）

**用途**：
- 新雇主入职要求
- 申请贷款
- 租房证明

**如公司拒绝提供**：
- 向 OMMF 投诉
- 公司可能被罚款

## 4. 赔偿计算详解

### 4.1 法定赔偿标准

| 工作年限 | 赔偿月数 |
|---------|---------|
| < 3 年 | 0 |
| 3-5 年 | 1 个月 |
| 5-10 年 | 2 个月 |
| 10-15 年 | 3 个月 |
| 15-20 年 | 4 个月 |
| 20-25 年 | 5 个月 |
| > 25 年 | 6 个月 |

### 4.2 工资计算基数

**包含项目**：
- 基本工资
- 固定津贴
- 过去 6 个月平均奖金

**计算公式**：
```
月平均工资 = (过去 6 个月总收入) / 6
赔偿金额 = 月平均工资 × 赔偿月数
```

**示例**：
```
工作年限：7 年
过去 6 个月工资：
- 基本工资：400,000 HUF/月
- 奖金平均：100,000 HUF/月
- 合计：500,000 HUF/月

赔偿月数：2 个月
赔偿金额：500,000 × 2 = 1,000,000 HUF
```

### 4.3 赔偿上限

**法定上限**：年薪的 6 倍
- 约 2024 年：18,000,000 HUF

**超过上限的处理**：
- 即使计算结果更高，也按上限支付
- 或协商其他补偿方式

## 5. 特殊情况处理

### 5.1 公司破产

**员工权益**：
- 优先获得赔偿
- 可向国家保险基金申请

**申请流程**：
1. 向破产管理人登记债权
2. 提供劳动合同、工资单
3. 等待分配

### 5.2 跨国调动

**如在跨国公司内部调动**：
- 可能涉及不同国家劳动法
- 需确认工龄是否连续计算
- 赔偿标准按最后工作地确定

### 5.3 签证/居留问题

**工作居留许可**：
- 离职后 30 天内需找到新工作
- 或申请其他类型居留
- 否则需离境

**建议**：
- 先确认新工作再辞职
- 或咨询移民局了解选项

## 6. 时间线总结

### 辞职时间线

```
Day 0:  提交辞职信
        ↓
Day 1-3: 等待公司确认
        ↓
Day 4-30: 通知期（可用年假抵扣）
        ↓
Day 30:  最后工作日
        ↓
Day 30+: 领取最终工资（通常次月 10 日）
```

### 解雇时间线

```
Day 0:   收到解雇通知
        ↓
Day 0-3: 决定是否接受/抗争
        ↓
Day 3-30: 通知期（如接受）
         或 咨询律师准备起诉（如抗争）
        ↓
Day 30:  最后工作日
        ↓
Day 30+: 领取赔偿 + 最终工资
```
