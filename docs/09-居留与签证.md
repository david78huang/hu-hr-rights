## 🛂 签证类型概览

### 短期签证 (C类，≤90天)

| 类型 | 用途 | 停留期 |
|-----|------|--------|
| **旅游签证** | 旅游、探亲 | 90天内 |
| **商务签证** | 商务会议、考察 | 90天内 |
| **探亲签证** | 探望亲属 | 90天内 |

### 长期签证 (D类，>90天)

| 类型 | 用途 | 有效期 |
|-----|------|--------|
| **学习签证** | 留学 | 1年，可续 |
| **工作签证** | 受雇工作 | 1-2年，可续 |
| **家庭团聚** | 与亲属团聚 | 与担保人相同 |
| **访客投资者** | 黄金签证 | 10年 |

## 💼 工作许可 (Munkavállalási engedély)

### 申请条件

**1. 有雇主担保**
- 匈牙利雇主提供工作合同
- 雇主向劳动局申请工作许可

**2. 高技能工作者（欧盟蓝卡）**
- 大学学历
- 年薪达到一定标准（约60,000€/年）
- 无需劳动局审批，速度更快

**3. 短缺职业**
- 匈牙利短缺职业清单上的职位
- 审批更快、要求更宽松

### 申请流程

```
1. 找到雇主
   ├── 签订工作合同
   └── 雇主准备申请材料

2. 雇主申请工作许可
   ├── 向劳动局（OIF）提交申请
   ├── 证明无本地/欧盟候选人
   └── 等待审批（通常30-60天）

3. 申请D类签证
   ├── 向中国使领馆提交申请
   ├── 工作许可获批后
   └── 等待签证（通常15-30天）

4. 入境后办理居留卡
   ├── 入境30天内
   ├── 向移民局申请
   └── 领取居留卡（通常30天）
```

### 所需材料

| 材料 | 说明 |
|-----|------|
| 护照 | 有效期覆盖居留期 |
| 工作合同 | 匈牙利雇主签署 |
| 工作许可 | 劳动局批准 |
| 住宿证明 | 租房合同或酒店预订 |
| 资金证明 | 银行存款或工资证明 |
| 保险 | 医疗保险 |
| 照片 | 证件照 |
| 申请表 | 使领馆提供 |

## 🏠 居留卡 (Tartózkodási kártya)

### 首次申请

**申请条件：**
- 持有D类签证入境
- 有合法居留理由（工作、学习、家庭等）
- 有稳定住所
- 有医疗保险

**申请流程：**
1. 入境后30天内申请
2. 前往移民局（Bevándorlási és Menekültügyi Hivatal）
3. 提交材料、录指纹
4. 等待制卡（通常30天）
5. 领取居留卡

### 居留卡续签

**续签时间：**
- 到期前30-45天申请
- 建议提前2个月准备

**续签材料：**
- 现有居留卡
- 护照
- 继续居留理由证明（工作合同/在读证明等）
- 住宿证明
- 保险证明
- 资金证明

### 居留卡类型

| 类型 | 有效期 | 说明 |
|-----|--------|------|
| **工作居留** | 1-2年 | 与合同期限挂钩 |
| **学习居留** | 1年 | 与学制挂钩 |
| **家庭团聚** | 1-3年 | 与担保人相同 |
| **访客投资者** | 10年 | 黄金签证 |

## 👨‍👩‍👧 家庭团聚签证

### 申请条件

**担保人资格：**
- 匈牙利公民，或
- 持有匈牙利居留卡的外国人

**可申请团聚的亲属：**
- 配偶
- 未成年子女（18岁以下）
- 受抚养的成年子女
- 受抚养的父母

### 申请流程

```
1. 担保人准备材料
   ├── 收入证明（达到最低标准）
   ├── 住宿证明
   └── 关系证明

2. 被担保人申请签证
   ├── 向中国使领馆提交
   ├── 家庭团聚签证
   └── 等待审批

3. 入境后申请居留卡
   └── 流程与普通居留卡相同
```

### 收入要求

担保人需证明有足够收入支持家庭成员：
- 配偶：约150,000福林/月
- 每个子女：约100,000福林/月

## 🏅 黄金签证（访客投资者居留）

### 2024年新政策

匈牙利于2024年推出新的黄金签证计划，称为"访客投资者居留"。

### 投资选项

| 选项 | 金额 | 说明 |
|-----|------|------|
| **房地产基金** | 25万欧元 | 投资政府认可的房地产基金 |
| **住宅房产** | 50万欧元 | 购买匈牙利住宅房产 |
| **慈善捐赠** | 100万欧元 | 捐赠给指定的公益信托 |

### 居留权益

- **有效期**：10年
- **可续签**：再续10年
- **工作权利**：可在匈牙利工作
- **家庭团聚**：配偶和未成年子女可一同申请
- **欧盟通行**：可免签进入申根区

### 申请流程

```
1. 准备投资
   ├── 选择投资方式
   ├── 完成资金准备
   └── 进行合规投资

2. 申请访客投资者签证
   ├── 向中国使领馆提交
   ├── 提供投资证明
   └── 等待审批

3. 入境完成投资
   ├── 90天内完成投资
   └── 获得投资证明

4. 申请居留许可
   ├── 向移民局提交
   ├── 提供投资完成证明
   └── 等待审批（通常60天）

5. 获得10年居留卡
```

### 优势

- 无居住要求（无需每年住满多少天）
- 无语言要求
- 无学历要求
- 可带全家
- 10年长期稳定

## 🏛️ 永久居留 (Állandó tartózkodási engedély)

### 申请条件

**连续合法居留3年：**
- 持有居留卡连续3年
- 每年离境不超过90天
- 有稳定收入
- 有医疗保险
- 通过基础匈牙利语考试（部分情况豁免）

**或特殊情况：**
- 与匈牙利公民结婚3年以上
- 在匈牙利出生并居住
- 对匈牙利有特殊贡献

### 所需材料

- 护照及居留卡
- 收入证明
- 住宿证明
- 保险证明
- 语言证明（如适用）
- 无犯罪记录

### 永久居留权益

- 无限期居留（无需续签）
- 自由工作
- 享受社会福利
- 可申请入籍（符合条件后）

---

## 🇭🇺 入籍 (Magyar állampolgárság)

### 申请条件

**普通入籍：**
- 连续居住8年
- 通过匈牙利语考试
- 通过宪法考试
- 无犯罪记录
- 有稳定收入

**简化入籍：**
- 与匈牙利公民结婚3年
- 在匈牙利出生
- 祖先为匈牙利公民

### 入籍流程

1. 准备材料
2. 向区政府提交申请
3. 参加语言考试
4. 参加宪法考试
5. 等待审批（通常6-12个月）
6. 宣誓入籍

## 🆘 常见问题

### Q: 工作许可和居留卡有什么区别？
**A:** 工作许可是工作权利，居留卡是居住权利。通常先申请工作许可，再申请居留卡。

### Q: 居留卡到期前多久申请续签？
**A:** 建议到期前30-45天申请，最晚不要晚于到期日。

### Q: 换工作需要重新申请居留卡吗？
**A:** 需要。新雇主需申请新的工作许可，然后更新居留卡。

### Q: 黄金签证的投资可以撤回吗？
**A:** 投资需保持至少5年，提前撤回可能影响居留权。

### Q: 永久居留和入籍有什么区别？
**A:** 永久居留是无限期居住权，入籍是成为匈牙利公民（获得护照和选举权）。

## 💡 小贴士

1. **提前规划** - 签证和居留申请耗时，提前2-3个月准备
2. **保留记录** - 保存所有出入境记录、居住证明
3. **及时续签** - 不要等到最后一刻才申请续签
4. **专业咨询** - 复杂情况建议咨询移民律师
5. **关注政策** - 移民政策可能变化，关注官方信息
