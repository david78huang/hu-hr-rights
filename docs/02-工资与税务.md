## 1. 工资结构

### 1.1 工资单常见项目

| 项目（匈语） | 中文 | 说明 |
|------------|------|------|
| Bruttó bér | 税前工资 | 合同约定的名义工资 |
| Nettó bér | 税后工资 | 实际到手金额 |
| Szja | 个人所得税 | 15% 统一税率 |
| Tb járulék | 社保缴纳 | 18.5%（员工部分）|
| Nyugdíjjárulék | 养老金 | 包含在社保中 |
| Egészségbiztosítás | 医疗保险 | 包含在社保中 |

### 1.2 工资计算示例

**示例：月薪 500,000 HUF**

```
税前工资 (Bruttó):           500,000 HUF

扣除项目：
- 个人所得税 (15%):          -75,000 HUF
- 社保缴纳 (18.5%):          -92,500 HUF

税后工资 (Nettó):            332,500 HUF

实际到手率: 66.5%
```

### 1.3 雇主额外成本

雇主除了支付税前工资，还需额外缴纳：
- **社保雇主部分**: 约 13%（具体因行业而异）
- **职业培训基金**: 1.5%

**总成本示例**：
```
员工税前工资: 500,000 HUF
雇主社保缴纳: +65,000 HUF (约13%)
培训基金:     +7,500 HUF (1.5%)

雇主总成本:   572,500 HUF
```

## 2. 个人所得税 (Személyi jövedelemadó)

### 2.1 税率

匈牙利个人所得税采用 **统一税率制**：

| 收入类型 | 税率 |
|---------|------|
| 工资收入 | 15% |
| 奖金收入 | 15% |
| 加班费 | 15% |

**特点**：
- 无累进税率，高收入者也只需缴纳 15%
- 无免税额度（2021年起取消）

### 2.2 税收减免

虽然税率统一，但部分支出可抵税：

| 减免项目 | 年度上限 | 说明 |
|---------|---------|------|
| 家庭税收减免 | 根据子女数量 | 有子女家庭可享受 |
| 养老金储蓄 | 最高 150,000 HUF/年 | 自愿养老金账户 |
| 住房储蓄 | 最高 100,000 HUF/年 | 购房/建房用途 |
| 捐赠 | 收入的 30% | 给公益组织 |

**家庭税收减免示例**：
```
有 1 个子女：每月减免约 10,000 HUF
有 2 个子女：每月减免约 20,000 HUF
有 3 个及以上：每月减免约 33,000 HUF
```

### 2.3 如何申请减免

1. **向雇主提交申请**（最常见）
   - 填写 "Adóelőny kérelem" 表格
   - 提供子女出生证明等文件
   - 雇主在发工资时直接扣除

2. **年度汇算**（次年 3-5 月）
   - 通过 NAV 网站在线申报
   - 或委托税务顾问处理
   - 多缴税款将在 30 天内退还

## 3. 社会保险 (Társadalombiztosítás)

### 3.1 社保组成

| 保险类型 | 员工缴纳 | 雇主缴纳 | 合计 |
|---------|---------|---------|------|
| 养老金 (Nyugdíj) | 10% | 0%* | 10% |
| 医疗保险 (Egészségügy) | 7% | 0%* | 7% |
| 失业保险 (Munkanélküli) | 1.5% | 0%* | 1.5% |
| **员工合计** | **18.5%** | - | - |
| 雇主社保 | - | **13%** | **13%** |

*注：2021年改革后，养老金和医疗保险的雇主部分已并入统一税率，不再单独显示。*

### 3.2 社保权益

**医疗保险覆盖**：
- 公立医院免费治疗
- 基础药物补贴
- 年度体检
- 牙科基础治疗

**养老金权益**：
- 需缴纳满 20 年可领取全额养老金
- 退休年龄：男性 65 岁，女性根据出生年份 62-65 岁不等

**失业保险**：
- 失业后可领取最多 90 天失业救济金
- 金额为之前工资的 60%

### 3.3 社保号码 (TAJ szám)

- 格式：9 位数字（如：123 456 789）
- 作用：就医、领药、查询社保记录
- 办理：入职时由雇主统一申请

**查询社保缴纳记录**：
- 网站：https://eszja.nav.gov.hu/
- 需要：税务号码 + 密码

## 4. 华人常见问题

### 4.1 工资应该报多少税？

**常见做法**：
| 方式 | 说明 | 风险 |
|-----|------|------|
| 全额报税 | 按实际工资缴税 | 无风险，推荐 |
| 部分现金 | 部分工资现金支付，不报税 | 被查出需补税+罚款，影响居留 |

**⚠️ 提醒**：
- 现金工资无法证明收入，影响贷款、租房
- 社保缴纳记录与养老金挂钩
- 税务局抽查越来越严格

### 4.2 如何查询自己的税务情况？

**在线查询**：
1. 访问 https://eszja.nav.gov.hu/
2. 用税务号码登录
3. 查看年度收入汇总

**手机 App**：
- NAV 官方 App（匈牙利语）
- 可查看每月缴税记录

### 4.3 换工作时税务如何处理？

**无缝衔接**：
- 新雇主会自动继承税务信息
- 无需个人操作

**有间隔期**：
- 年度汇算时统一处理
- 可能涉及补税或退税

### 4.4 年度汇算（Adóbevallás）

**时间**：每年 3 月 1 日 - 5 月 20 日

**谁需要申报**：
- 有额外收入（房租、投资收益等）
- 要申请税收减免
- 有多份工作

**如何申报**：
1. **在线申报**（推荐）：https://eszja.nav.gov.hu/
2. **委托会计师**：费用约 10,000-30,000 HUF
3. **雇主代办**：部分雇主提供免费服务

## 5. 实用工具

### 5.1 工资计算器

在线工具：
- https://www.bérkalkulátor.hu/（匈牙利语）
- 输入税前工资，自动计算税后金额

### 5.2 关键时间节点

| 时间 | 事项 |
|-----|------|
| 每月 10 日 | 雇主缴纳上月社保 |
| 每月 12 日 | 雇主代扣个税上缴 |
| 每年 1 月 | 雇主发放年度收入证明 |
| 3-5 月 | 年度税务申报期 |
