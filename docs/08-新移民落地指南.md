## 🏠 地址卡（Lakcímkártya）

### 什么是地址卡？

地址卡是匈牙利居民的**法定居住证明**，包含你的姓名、出生日期和匈牙利居住地址。几乎所有行政手续都需要它。

### 办理条件

- 有效护照
- 匈牙利居住地址（租房合同或房产证明）
- 房东同意书（由房东填写）

### 办理流程

```
1. 准备材料
   ├── 护照原件及复印件
   ├── 租房合同原件
   └── 房东填写的地址登记表

2. 前往区政府（Kormányablak 或 Önkormányzat）
   ├── 预约或现场取号
   ├── 提交材料
   └── 等待制作（约30分钟-2小时）

3. 领取地址卡
   └── 当场领取或邮寄到家
```

### 所需材料清单

| 材料 | 说明 | 备注 |
|-----|------|------|
| 护照 | 原件及复印件 | 有效期需覆盖居住期 |
| 租房合同 | 原件 | 需房东签字 |
| 地址登记表 | 房东填写 | 表格可在区政府领取 |
| 申请费 | 约1000福林 | 以实际收费为准 |

## 💰 税号（Adószám）

### 什么是税号？

税号是匈牙利税务系统的**个人识别号码**，用于：
- 工资发放和纳税
- 银行开户
- 签订合同
- 社保缴纳

### 办理流程

```
1. 在线申请（推荐）
   ├── 访问 https://nav.gov.hu/
   ├── 填写T34表格
   └── 上传护照扫描件

2. 或现场办理
   ├── 前往税务局（NAV）
   ├── 填写T34表格
   └── 提交护照复印件

3. 等待审批
   └── 约1-2周收到税号
```

### T34表格填写要点

| 字段 | 填写内容 | 备注 |
|-----|---------|------|
| Név / Name | 护照上的英文姓名 | 与护照一致 |
| Születési név | 出生姓名 | 如有变更 |
| Anyja neve | 母亲姓名 | 匈牙利习惯 |
| Születési hely, idő | 出生地点、日期 | |
| Állampolgárság | 国籍 | CHINESE |
| Lakcím | 匈牙利地址 | 需与地址卡一致 |

| 网站 | https://nav.gov.hu/ |
| 电话 | +36 1 327 3100 |
| 办公时间 | 周一至周五 8:00-16:00 |

## 🏥 TAJ医保号

### 什么是TAJ？

TAJ（Társadalombiztosítási Azonosító Jel）是匈牙利**社会医疗保险号**，是享受公共医疗服务的凭证。

### 申请条件

- 合法居留
- 有工作合同（雇主缴纳社保）或
- 自费参保（每月约8000福林）

### 申请流程

```
1. 有工作的情况
   ├── 雇主在雇佣时自动申请
   └── 约2-4周收到TAJ卡

2. 自费参保
   ├── 前往社保局（OEP/NEAK）
   ├── 填写申请表
   ├── 缴纳首月保费
   └── 等待TAJ卡邮寄
```

### 社保局联系方式

|-----|------|
| 网站 | https://www.neak.gov.hu/ |
| 电话 | +36 1 327 3400 |

## 🆘 常见问题

### Q: 没有地址卡可以开银行账户吗？
**A:** 部分银行允许先用护照开户，但需在30天内补交地址卡。建议先办地址卡。

### Q: 税号和TAJ号有什么区别？
**A:** 税号用于税务，TAJ号用于医疗。两者独立，都需要申请。

### Q: 地址卡可以写朋友家地址吗？
**A:** 可以，但需朋友作为房东签署同意书，且可能涉及税务问题。建议用自己的租房合同。

### Q: 办理这些手续需要会说匈牙利语吗？
**A:** 地址卡和银行通常有英文服务，税务局建议带翻译或使用在线申请。
